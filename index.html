About Me

Halo nama saya Aan Rehan, bisa dipanggil Rehan. Saya mempunyai ketertarikan di bidang Penetration Testing Web , website ini isinya adalah semua catatan saya selama menjadi BUG Hunter, dan akan saya update secara berkala jika mendapatkan temuan baru lagi.

Saya adalah siswa SMK Pertanian, ketertarikan utama saya adalah segala sesuatu yang berhubungan dengan Penetration Testing. Karena ketertarikan tersebut, seringkali saya menghabiskan waktu yang saya miliki hanya untuk membaca tentang perkembangan teknologi yang terjadi setiap harinya dan sesekali mencoba langsung jika itu adalah sesuatu yang berhubungan dengan issue security web application.
